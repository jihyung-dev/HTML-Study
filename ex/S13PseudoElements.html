CSS 의사 요소 선택자

  • ::before - 요소의 내용 앞에 가상의 콘텐츠를 추가
  • ::after - 요소의 내용 뒤에 가상의 콘텐츠를 추가
  • ::first-letter - 첫 번째 글자에 스타일 적용
  • ::first-line - 첫 번째 줄에 스타일 적용
  • ::selection - 사용자가 선택한 텍스트에 스타일 적용
  • ::marker - 목록 불릿이나 번호에 스타일 적용

사용 예시

  • p::before { content: "★ "; color: gold; } - 문단 앞에 별 추가
  • p::after { content: " (끝)"; color: gray; } - 문단 뒤에 '(끝)' 추가
  • p::first-letter { font-size: 2em; } - 첫 글자 확대
  • p::first-line { color: blue; } - 첫 줄 색상 변경
  • ::selection { background: yellow; } - 선택 영역 배경 노랑
  • li::marker { color: red; } - 리스트 불릿 빨강

주의사항

  • ::before, ::after는 반드시 content 속성을 지정해야 표시됨
  • CSS3부터는 :: (더블 콜론) 표기를 권장
  • 의사 요소는 HTML에 없는 가상의 요소에 스타일 적용
