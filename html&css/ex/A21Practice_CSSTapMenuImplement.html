순수 CSS 탭 메뉴 구현하기

  • 라디오 버튼 + label을 이용하여 탭을 전환한다. label을 클릭하면 연결된 input[type="radio"]가 체크되며, 이를 CSS로 감지해 콘텐츠를 토글한다.
  • :has() 선택자를 활용해 “특정 라디오 버튼이 체크되었을 때 해당 콘텐츠를 보이게” 만든다.
  • 애니메이션은 display:none 대신 max-height, opacity , transform을 조합해 부드럽게 펼침/접힘 효과를 만든다.
