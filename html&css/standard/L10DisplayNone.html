다양한 방법으로 요소를 숨겨봅시다.

  • hidden :
    hidden은 input만 사라짐
  • display:none
    (요소가 없는 것처럼 취급됨)
  • opacity (투명도) : 0
    (보이지만 않음)
  • width: 0
    컨텐츠
    요소의 크기를 0으로 만들어도 컨텐츠는 없어지지않음
  • width: 0; overflow: hidden;
    컨텐츠
    넘쳐흐르는 컨텐츠를 hidden으로 감춘다.
  • border가 존재하면 크기로 사라지게 하지 못함
  • box-sizing: border-box로 해도 크기가 무관하게 border가 생성 (무조건 border-width=0으로 지정, border:none)
  • visibility: hidden (default: visible)
    요소는 존재하지만 보이지 않게 하는 방법
  • transform:scale(0) :
    transform을 이용해 제거 실제로 줄었지만, 공간은 남음,

의사 클래스 (상태)
