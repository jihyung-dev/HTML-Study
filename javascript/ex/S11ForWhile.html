반복문 for while

반복문(Loop)이란?

  • 반복문은 특정 코드 블록을 여러 번 실행하기 위해 사용
  • 조건이 true인 동안 반복하거나, 정해진 횟수만큼 반복 가능
  • 프로그램에서 같은 작업을 효율적으로 처리할 수 있게 함

반복문을 사용하는 목적

  • 동일한 코드를 여러 번 작성하지 않고, 한 번의 정의로 반복 실행
  • 대량의 데이터 처리(예: 배열의 모든 값 출력)
  • 조건을 만족할 때까지 작업 반복

반복문의 종류

1. for 문

  • 반복 횟수가 정해져 있을 때 사용
  • 형식: for (초기식; 조건식; 증감식) { ... }
for (let i = 0; i < 5; i++) {
  console.log(i); // 0,1,2,3,4
}

2. while 문

  • 조건이 true인 동안 반복
  • 반복 횟수를 알 수 없을 때 사용
let n = 0;
while (n < 3) {
  console.log(n);
  n++;
}

3. do ~ while 문

  • 조건을 나중에 검사 → 최소 한 번은 실행
let x = 0;
do {
  console.log(x);
  x++;
} while (x < 3);

4. for ... of 문

  • 배열, 문자열 등 iterable 객체의 값을 순회
let fruits = ["사과", "바나나", "체리"];
for (let fruit of fruits) {
  console.log(fruit);
}

5. for ... in 문

  • 객체의 키(속성 이름)을 순회
let user = { name: "경민", age: 39 };
for (let key in user) {
  console.log(key, ":", user[key]);
}

반복문 제어

  • break: 반복 즉시 종료
  • continue: 현재 반복 건너뛰고 다음 반복 진행
for (let i = 1; i <= 5; i++) {
  if (i === 3) continue; // 3 건너뜀
  if (i === 5) break;    // 5에서 종료
  console.log(i); // 1,2,4 출력
}

반복문 연습 문제 & 정답

기본 문제 정답

  1. for문을 이용하여 1부터 10까지 숫자를 출력 
    for (let i = 1; i <= 10; i++) {
  console.log(i);
}
  2. while문을 사용하여 5부터 1까지 역순 출력 
    let n = 5;
while (n >= 1) {
  console.log(n);
  n--;
}
  3. do~while문을 이용하여 0부터 2까지 출력 
    let x = 0;
do {
  console.log(x);
  x++;
} while (x <= 2);

배열과 반복문 정답

  1. for...of문으로 배열 값 출력 
    let cities = ["서울", "부산", "대구"];
for (let city of cities) {
  console.log(city);
}
  2. for...in문으로 객체 속성과 값 출력 
    let book = { title: "책", price: 15000 };
for (let key in book) {
  console.log(key, ":", book[key]);
}

조건과 제어 정답

  1. for문으로 1~10 중 짝수만 출력 
    for (let i = 1; i <= 10; i++) {
  if (i % 2 === 0) {
    console.log(i);
  }
}
  2. continue 사용하여 3 건너뛰기 
    for (let i = 1; i <= 5; i++) {
  if (i === 3) continue;
  console.log(i);
}
  3. break 사용하여 6에서 멈추기 
    for (let i = 1; i <= 10; i++) {
  if (i === 6) break;
  console.log(i);
}

응용 문제 정답

  1. 1부터 100까지 합 구하기

for (let i = 1; i <= 100; i++) {

let num = 1;
while (total < 50) {

    console.log("누적합이 50 이상이 된 숫자:", num);

  num++;
