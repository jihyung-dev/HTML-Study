JS반복자와 Array 반복문

  • 문제 1. 배열 [1,2,3,4,5]를 for문으로 순회하며 모든 요소를 출력하시오.
  • 문제 2. 배열 ["사과","바나나","포도"]를 for...of문으로 순회하며 과일 이름을 출력하시오.
  • 문제 3. 배열 ["a","b","c"]를 for...in문으로 순회하며 인덱스와 값을 함께 출력하시오.
  • 문제 4. 배열 [10,20,30]을 forEach로 순회하며 각 요소에 "점수:"를 붙여 출력하시오.
  • 문제 5. 배열 [1,2,3,4,5]에서 map을 이용해 각 요소를 제곱한 새 배열을 만드시오.
  • 문제 6. 배열 [12,5,8,130,44]에서 filter를 이용해 10 이상인 값만 추출하시오.
  • 문제 7. 배열 [1,2,3,4]의 합을 reduce로 구하시오. (초기값은 0)
  • 문제 8. 배열 [1,3,5,7,9]에서 some을 이용해 짝수가 있는지 확인하시오.
  • 문제 9. 배열 [2,4,6,8]에서 every를 이용해 모든 값이 짝수인지 확인하시오.
  • 문제 10. 배열 ["철수","영희","민수"]에서 find를 이용해 "영희"를 찾고, findIndex를 이용해 그 인덱스를 출력하시오.
